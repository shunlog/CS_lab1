# CS lab 1: Caesar cipher

## Usage

``` sh
$ ./caesar.py -h 
usage: Caesar cipher [-h] [--decrypt] --key1 KEY1 [--key2 KEY2] [--info] text

Encrypt or decrypt a text using the Caesar cipher

positional arguments:
  text

options:
  -h, --help            show this help message and exit
  --decrypt, -d
  --key1 KEY1, -k1 KEY1
  --key2 KEY2, -k2 KEY2
  --info, -i
```
